# MyLog

**🔗 데모 URL : https://mylog-437d4.web.app/**

## 1. 개요
글을 업로드하고 다른 사람의 글을 확인할 수 있는 블로그 서비스입니다. 구글 회원가입 및 로그인으로 서비스를 이용할 수 있어 안정적이고 접근이 용이하도록 구성하였습니다. 로그인, 글 작성, 글 확인 기능을 제공합니다.


## 2. 기술 스택
- FrontEnd: React, HTML5, Tailwind CSS, JavaScript
- BackEnd: Firebase
